# Project Context

## Overview
Personal portfolio + blog website for Charmayne, a founding designer, growth marketer, and builder.

## Goals
- Showcase 2 project case studies
- Host personal blog ("log / writing")
- Minimal, clean aesthetic with monospace typography
- Fast, static site with excellent performance
- Easy content updates via markdown files

## Constraints
- Static site generation (no backend)
- No frameworks for styling (vanilla CSS)
- Must be responsive (mobile, tablet, desktop)
- Browser support: Modern evergreen browsers only

## Technology Stack
- **Framework:** Astro (static site generation)
- **Content:** Markdown with frontmatter
- **Styling:** Vanilla CSS with CSS custom properties
- **Fonts:** Inter, Geist Mono, Spectral (self-hosted or Google Fonts)
- **Deployment:** Vercel / Netlify / Cloudflare Pages

## Target Audience
Potential clients, collaborators, and readers interested in product design, growth marketing, and building.

## Success Criteria
- All pages load in <1s
- Perfect Lighthouse scores (Performance, Accessibility, SEO)
- Easy to add new blog posts (markdown file + frontmatter)
- Clean, professional design matching Figma specs
