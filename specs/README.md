## Quick Start

To implement this project, follow these steps:

1. Read through all spec files to understand the full scope
2. Follow the implementation guide in `06-implementation.md`
3. Refer back to individual spec files for detailed requirements

## Technology Stack

- **Framework:** Astro (static site generation)
- **Content:** Markdown with frontmatter
- **Styling:** Vanilla CSS with CSS custom properties
- **Fonts:** Inter, Geist Mono, Spectral
- **Deployment:** Vercel / Netlify / Cloudflare Pages

## Key Features

- Portfolio homepage with bio and project links
- 2 project case study pages
- Blog index and individual blog post pages
- Minimal, clean aesthetic with monospace typography
- Fully responsive (mobile, tablet, desktop)
- Fast, static site with excellent performance
